USERS

Method PATH DESCRIPTION
GET /users Get all users
GET /users/:id Get a single user
POST /users/register Register A user
PATCH /users/update/:id Update A User
POST /users/login Login A USER
DELETE /users/delete/:id

ITEMS

Method PATH DESCRIPTION
GET /items Get all items
GET /items/:id Get a single item
POST items/register Register A New Item
PATCH items/update/:id Update item info
DELETE items/delete/:id Remove Item

ORDERED ITEMS

Method PATH DESCRIPTION
GET /users/usersID/orders/ Get Items ordered
GET /users/:usersID/orders/:orderID View A single order
POST /users/:usersID/order Complete Order
PATCH /users/:usersID/order/update/:orderID Edit Ordered Item
DELETE /users/:usersID/orders/delete Delete All Orders
DELETE /users/:usersID/orders/delete/:orderID Delete A Single Order
